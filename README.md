# Vitalis
Designed to be a fitness chat bot, improve yourself with the new AI

# Crear entorno virtual
```bash
python -m venv venv
# Activar el entorno en linux
source venv/bin/activate
# En windows
.\venv\Scripts\Activate.ps1
```
# Requirements
```bash
pip install reflex
pip install --upgrade pip
```
# Iniciar
```bash
reflex init
reflex run
```
